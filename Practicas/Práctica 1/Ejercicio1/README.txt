Programa Cliente-Servidor que permite el intercambio de mensajes de texto

Programas que se deben ejecutar:
  -Servidor: Servidor_Socket.java
  -Cliente: Cliente.c

El servidor se compila con "javac Servidor_Socket.java" y se ejecuta con "java Servidor_Socket"
El cliente se compila con "gcc Cliente.c -o Cliente" y se ejecuta con "./Cliente 127.0.0.1"
